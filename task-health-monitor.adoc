= Monitor health of NetApp Data Classification
:hardbreaks:
:nofooter
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
The NetApp Data Classification Health Monitor dashboard provides real-time monitoring and insights into your Data Classification infrastructure, system information, usage metrics, and utilization data. Use the Health Monitor to optimize performance and storage management. 

== Health Monitor insights 

The Health Monitor dashboard presents information in four categories. 

* Infrastructure status
** Version status
** System stability
** Deployment type
** Machine scale

* Problematic containers
+
Review the problematic containers field for insights into containers that are stopped or restarting frequently. Use this information to investigate the specific containers. 

* System information 
+
The system information panel captures critical information about the NetApp Console and Data Classification such as the public and private IP addresses, host name, operating system, Console version, and Console ID. 

* Usage and utilization
+
Review the CPU usage, disk utilization, disk usage, and memory usage. These values are displayed in storage units (GB) or percentages of total use. If any fields display warnings, select the warning for more detailed information and remediation recommendations. 

== Access the Health Monitor dashboard

. In Data Classification, select **Configuration**.
. Under the **Configuration** heading, select **Data Classification health monitor**.
. In the Health Monitor dashboard, you can:
* Review the usage and utilization. If any usage or utilization metrics are presenting warnings, select the field for recommendations to ameliorate the situation. 
* Toggle the graph to display CPU usage, disk utilization, disk usage, and memory usage. You can change the x-axis to display content over hours (6, 12, or 24) or days (2, 7, or 14).
* Refresh the dashboard for more up-to-date insights into your data.
